# DESCRIPTION:

This Real-Time Chat Application is developed as a part of my Front-End Development internship at CodTech IT Solutions.  
The primary objective of this task is to build a live messaging platform that allows users to send and receive messages instantly without reloading the page.

This web application uses **Socket.io** for real-time communication, **Node.js** for backend server handling, and **HTML, CSS, and JavaScript** for the user interface.

The project demonstrates how WebSockets can enable seamless two-way data communication between a browser and a server, which is essential in modern chat and messaging applications. This real-time feature ensures fast communication and an interactive chat experience.

## KEY FEATURES:

• Real-time message sending and receiving using Socket.io

• Automatically displays the messages instantly on all connected clients 

• Simple and clean UI for user-friendly communication  

• Separate frontend and backend folder structure following real project standards 

• Scalable architecture that supports multiple users  

• Lightweight and fast performance  

• Completely built using open-source technologies  

## WORKING PRINCIPLE:

• A user opens the chat app in a browser  
• When a message is typed and sent, the message travels through a **WebSocket connection**  
• The server receives the message and **broadcasts** it to all connected clients  
• All users see the message instantly without refreshing the page

This interaction is powered by **Socket.io**, which handles socket creation, event listening, and broadcasting.
